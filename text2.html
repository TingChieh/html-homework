Raspberry Pi 设置 frp

How to set

树莓派设置sakura frp开机自启动

 

首先创建文件/etc/systemd/system/sakurafrp.service 在sakurafrp.service中输入以下内容

[Unit]
Description=Sakura Frp Client
After=network-onine.target
[Service]
User=用户
WorkingDirectory=软件所在的文件夹
LimitNOFILE=4096
PIDFile=/var/run/sakurafrp/client.pid
ExecStartPre=sleep 60
ExecStart=软件所在的位置 --su=账号 --sp=密码 --sid=服务器序号
Restart=on-failure
StartLimitInterval=600
[Install]
WantedBy=multi-user.target

接着输入以下命令：

systemctl daemon-reload

服务就创建成功了，接下来启动服务：

systemctl start sakurafrp

将服务设置为开机启动

systemctl enable sakurafrp

如果要停止运行客户端，只需要输入

systemctl stop sakurafrp

如果要禁止开机启动，输入

systemctl disable sakurafrp
